# RingEngine

RingEngine is an open-source project focused on building a modern filter engine and filter editor for real-time visual workflows.

We want the whole pipeline to be open, inspectable, and practical:

- filter source standards and documentation
- schema-driven manifests and validation
- compiler and packaging toolchains
- a web-based editor and authoring experience
- runtime-facing formats that stay versioned and reproducible

Our current work is centered around the filter source specification, compiler behavior, schema alignment, package format design, and the early editor/tooling foundation.

The long-term goal is to make filter creation more reliable and more shareable, so creators can move from writing source files to validating, compiling, packaging, editing, and shipping filters with a clear and open standard.

## What We Are Building

- An open filter source standard
- A versioned runtime-compatible filter format
- Compiler tooling for validation and artifact generation
- Packaging and signing workflows for filter distribution
- A web editor for authoring and iteration

## Current Repositories

- `Docs`: standards, schemas, and specification documents
- `compiler-js`: compiler core, packer, and CLI tooling

## Vision

RingEngine aims to become a solid open foundation for building, validating, packaging, and editing filters across tools and runtimes.

# RingEngine

RingEngine 是一个开源项目，致力于构建现代化的滤镜引擎与滤镜编辑器，面向实时视觉处理与创作流程。

我们希望把整条链路做成开放、可检视、可落地的体系，包括：

- 滤镜源码标准与文档
- 基于 schema 的 manifest 约束与校验
- 编译与打包工具链
- 面向创作的 Web 编辑器
- 面向 runtime 的版本化产物格式

目前项目的重点在于滤镜源码规范、编译器行为、schema 对齐、资源包格式设计，以及编辑器和工具链的基础能力建设。

长期来看，我们希望让滤镜创作从源码编写、校验、编译、打包、编辑到发布，形成一套清晰、开放、可持续演进的标准化流程。

## 正在构建的内容

- 开放的滤镜源码标准
- 可版本化、可对齐 runtime 的滤镜格式
- 用于校验与产物生成的编译工具链
- 面向发布流程的打包与签名机制
- 用于创作与迭代的 Web 编辑器

## 当前仓库

- `Docs`：标准、schema 与规范文档
- `compiler-js`：编译器核心、打包器与 CLI 工具

## 愿景

RingEngine 希望成为一个可靠的开源基础设施，让滤镜的编写、校验、打包、分发与编辑都建立在清晰、公开的标准之上。
